Outlook Calendar
Paste your Outlook ICS link to sync your Exchange / Outlook calendar. Events update live.
To get your ICS link:
1. Go to outlook.office.com
2. Click ⚙️ Settings → Calendar → Shared calendars
3. Under "Publish a calendar" → select calendar → Can view all details → click Publish
4. Copy the ICS link (not the HTML link)

Connect Apple Health
Set up a one-time Shortcut so your steps sync automatically every time you open the app.
1. Create the Shortcut
Open the Shortcuts app → tap + → Name it "Hub Steps"
Add these 3 actions:
 ① Find Health Samples → Steps → Today
 ② Calculate Statistics → Sum
 ③ Open URLs → paste this URL:
2. Automate it
Go to Automation tab → + → App
Choose Safari (or your browser) → Is Opened
Add action: Run Shortcut → Hub Steps
Turn off "Ask Before Running"
✅ Once set up, steps will sync automatically every time you open the hub — no tapping needed.

Google Calendar
Connect your Google Calendar to see events here. You'll need a Google API Client ID.
Step 1 — Go to console.cloud.google.com, create a project and enable the Google Calendar API.
Step 2 — Create OAuth 2.0 credentials. Add your app's URL as an authorised origin.
Step 3 — Paste your Client ID below.
iCloud Calendar
Subscribe to a public iCloud calendar using its .ics link.
On your Mac/iPhone — Open Calendar → right-click a calendar → "Share Calendar" → enable Public Calendar → copy the webcal:// link.
Paste the link below. Replace webcal:// with https:// first. Events will be fetched and displayed read-only.
